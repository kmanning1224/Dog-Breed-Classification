Recognizing a dog's breed in an image is an easy task for human eyes. It is not too difficult to find dog breed pairs with inter-class variation, for instance, an American Bulldog and a Beagle. However, with a Brittany Spaniel and a Welsh Spaniel, it can be more difficult to differentiate the breeds given how similar they look in pictures.

Over the last decade, machine learning techniques and coding have made it easier to distinguish between dog breeds in images. We will walk you through how to create our machine learning technology and leverage the latest state of the art image classification techniques. Our model can be used as part of a mobile or web app, for real world, user-provided images. Give an image to the model, it determines if a dog is present and returns the estimated breed(s).
